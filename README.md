# Mnemikon Backend

Backend service for **Mnemikon**, an **Open Source Museum Collection Management System** built with Django, Django REST Framework and MongoDB.

Mnemikon Backend provides the REST API, authentication, business logic and data management required by the Mnemikon frontend application.

The project was designed as a modern, extensible and self-hosted solution for museums, cultural institutions and organizations that need to manage museum collections, research, restoration processes and collection movements.
